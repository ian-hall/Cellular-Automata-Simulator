Ian's Conway's Game Of Life

What was once a basic program written for CS101 has turned into this console implementation of Conway's Game Of Life. Written in C#, I have harnessed the power of the console to bring a robust, full-featured, retro-styled, buzzword described implementation of Conway's Game Of Life. I am currently working to add more support for 1D automata, as well as different rule sets for both 1D automata and 2D automata, such as Conway's. You can find it here on GitHub.

Just look at these awesome features:


Full-featured menus



Supports loading of CELLS-style files of populations



Advanced population creation functionality



Robust error handling



Now also supports 1D automata rules
