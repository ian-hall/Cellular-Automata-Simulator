Ian's cellular automata thing

I have harnessed the power of the console to bring a robust, full-featured, retro-styled, buzzword described program that lets users choose between types, rule sets, and starting populations. Users are able to utilize the advanced population creation feature to watch a custom 2D population grow or put in a custom hex string as a 1D rule. You can find it here on GitHub. You can even download a (maybe) working executable now.

Just look at these awesome features:


Advanced run controls



Support for CELLS files



"Robust" error handling



Many preset populations



Advanced population creation functionality



Psychedelic 1D rules
