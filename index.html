Ian's Conway's Game Of Life

What was once a basic program written for CS101 has turned into this console implementation of Conway's Game Of Life. You can find it here on GitHub. Just look at these awesome features:

Full-featured menus



Supports loading of CELLS-style files of populations



Advanced population creation functionality



Robust error handling



Now also supports 1D Automata rules
